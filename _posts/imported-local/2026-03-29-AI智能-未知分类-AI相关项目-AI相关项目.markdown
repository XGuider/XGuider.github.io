## 应用场景

### 爬虫 + ChatGPT 组合

```
爬取需要的数据 → 交给 ChatGPT 总结 → 发送给用户
```

典型应用流程：
1. 使用爬虫获取目标数据
2. 数据清洗和预处理
3. 调用 ChatGPT API 进行内容总结
4. 推送结果给用户
