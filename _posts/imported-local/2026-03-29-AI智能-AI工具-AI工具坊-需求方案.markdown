# AI 资源聚合平台需求文档

> 聚合全网免费资源，为大学生和职场人提供一站式资源发现平台

## 概述

本文档定义了 AI 资源聚合平台的核心需求，包括产品目标、用户权限、功能模块、技术实现和商业模式。

---

## 1. 核心目标

聚合全网合法公开的免费资源，为大学生/职场人提供一站式资源发现平台，解决资源分散问题。

---

## 2. 目标用户

| 用户类型 | 核心需求场景 |
|----------|--------------|
| 大学生 | 找免费学习资料、电子书、影视、AI 工具 |
| 职场人（初级） | 找技能课程、行业报告、效率工具 |

---

## 3. 核心功能

### 3.1 资源体系

| 维度 | 说明 |
|------|------|
| **资源类型** | 影视（按年份）、学习（按学科）、AI 工具（国内外整理）、电子书、付费软件（可扩展） |
| **分区设计** | 普通资源区（免费）、超级 VIP 区（稀缺资源） |

### 3.2 用户权限

| 用户类型 | 权限说明 |
|----------|----------|
| **游客** | 限次访问普通资源（如每日 3 次下载） |
| **普通会员** | 登录（微信一键授权）后无限访问普通资源 |
| **超级 VIP** | ¥10/月订阅，访问 VIP 区资源 |

### 3.3 管理后台

| 功能模块 | 说明 |
|----------|------|
| 资源管理 | 增删改/分类/审核 |
| 用户管理 | 封禁/VIP 开通 |
| 数据统计 | 访问量/VIP 转化率 |
| 留言系统 | 资源页面下方留言框 |

### 3.4 广告系统

普通资源区嵌入广告（如 Banner/信息流）

---

## 4. 技术实现

### 4.1 终端支持

| 终端 | 说明 |
|------|------|
| 微信小程序 | 主要入口 |
| Web 端 | 功能完全一致 |

### 4.2 登录方式

- 仅微信一键授权（无需注册）

### 4.3 合规要求

| 要求 | 说明 |
|------|------|
| 资源声明 | "来源公开免费"，禁止用户上传 |
| 隐私政策 | 仅收集微信公开信息 |

---

## 5. 商业模式

### 收入来源

| 来源 | 说明 |
|------|------|
| **超级 VIP 订阅** | ¥10/月（核心收入） |
| **广告收入** | 普通资源区（补充收入） |

### 收益预估

```
假设：1000 VIP 用户/月
月收入 = 1000 × 10 = ¥10,000

广告收入（假设 10 万 PV，RPM ¥5）
月广告收入 = 100,000 × 5 / 1000 = ¥500

总收入 ≈ ¥10,500/月
```

## 7. 关键业务规则

### 7.1 访问限制

| 用户类型 | 每日下载限制 |
|----------|--------------|
| 游客 | 3 次 |
| 普通会员 | 无限 |
| VIP | 无限 + VIP 资源 |

### 7.2 VIP 权益

- 无限访问所有资源
- 专属 VIP 资源区
- 无广告体验
- 优先客服支持

---

## 8. 风险与应对

| 风险 | 应对措施 |
|------|----------|
| 版权问题 | 仅收录公开免费资源，声明来源 |
| 内容审核 | 建立审核机制，用户举报 |
| 用户流失 | 持续更新资源，优化体验 |

---

## 9. 后续迭代方向

1. **社区功能**：用户评价、推荐系统
2. **积分系统**：签到、分享赚积分
3. **资源推荐**：个性化推荐算法
4. **多端支持**：App、桌面端
