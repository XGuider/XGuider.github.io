冒泡排序：原理比较两个想领的元素，将最大值交换到右边，依此的比较；N个数字排序需要

N-1躺排序，时间复杂度为O（n^2）

TopK的问题：

1、全排序，暴力O(n*n）

2、冒泡O(n*lg(k))

3、最小堆，python heapy和java的优先队列O(n*lg(k))

4、快排------核心思想：分治


分治法，大问题分解为小问题，小问题都要递归各个分支，例如：快速排序

减治法，大问题分解为小问题，小问题只要递归一个分支，例如：二分查找，随机选择

import heapq
class Solution(object):
 def findKthLargest(self, nums, k):
        pq=list()
        for i, num in enumerate(nums):
            if (i<k):
                heapq.heappush(pq,num)
                continue
            if num > pq[0]:
                heapq.heapreplace(pq, num)
        return  pq[0]

heapq的方法：
1、使用一个空列表，然后使用heapq.heappush()函数把值加入堆中
2、另外一种就是使用heap.heapify(list)转换列表成为堆结构
3、heapq.heappop() 函数弹出堆中最小值
4、删除堆中最小元素并加入一个元素，可以使用heapq.heaprepalce()
5、获取堆最大或最小值heapq.nlargest() 或heapq.nsmallest()可以设置范围和key的排序方式
6、heapq.heappushpop(h, n)加入并且弹出最小值

    private int partion(int[] nums, int start, int end) {
        int pivot = start, temp;
        while (start <= end) {
            while (start <= end && nums[start] <= nums[pivot])start++;
            while (start <= end && nums[end] > nums[pivot]) end--;
            if (start > end) break;
            temp = nums[start];
            nums[start] = nums[end];
            nums[end] = temp;
        }
        temp = nums[end];
        nums[end] = nums[pivot];
        nums[pivot] = temp;
        return end;
    }
}
