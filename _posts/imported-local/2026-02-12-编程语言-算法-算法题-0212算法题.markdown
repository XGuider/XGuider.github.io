题目编号： LeetCode 91
题目名称： 解码方法
难度： 🟠 中等
标签： 字符串、动态规划

一条包含字母 A-Z 的消息通过以下映射进行了 编码：
'A' -> "1"
'B' -> "2"
...
'Z' -> "26"
要 解码 已编码的消息，所有数字必须基于上述映射的方法，反向映射回字母（可能有多种方法）。
例如，"11106" 可以映射为：
"AAJAF"，将消息分组为 (1 1 10 6)
"KJF"，将消息分组为 (11 10 6)
注意，消息不能分组为 (1 11 06)，因为 "06" 不能映射为 "F"，这是由于 "6" 和 "06" 不同。
给你一个非空字符串 s，请计算并返回 解码 方法的 总数


示例 1：
输入： s = "12"
输出： 2
解释： 它可以解码为 "AB"（1 2）或者 "L"（12）。
示例 2：
输入： s = "226"
输出： 3
解释： 它可以解码为 "BZ" (2 26), "VF" (22 6), 或者 "BBF" (2 2 6) 。
示例 3：
输入： s = "06"
输出： 0
解释： "06" 无法映射到 "F"，因为存在前导零（"6" 和 "06" 不同）。


class Solution {
	public int numDecodings(String s) {
		if (s.length == 0 || s == '') {return 0;}
		if (s.length > 0 && s.charAt(0) == '0') {return 0;}
		int[] dp = new int[s.length];
		dp[0] = 1;

		for (int i = 1; i < s.length; i++) {
			if (s.charAt(i-1) <= '2' && s.charAt(i-1) > '0' && s.charAt(i) >= '0' && s.charAt(i) <= '9') {
				dp[i] = dp[i-1] + 1;
			} else {
				dp[i] = 1;
			}
		}
		return dp[s.length - 1];
	}
}

class Solution {
    public int numDecodings(String s) {
        if (s == null || s.isEmpty() || s.charAt(0) == '0') {
            return 0;
        }

        int n = s.length();
        int[] dp = new int[n + 1]; // dp[i] 表示前 i 个字符的解码方法数

        // 初始状态：空字符串有一种解码方式
        dp[0] = 1;
        // 第一个字符如果不是 '0'，则有一种解码方式
        dp[1] = 1;

        for (int i = 2; i <= n; i++) {
            // 单字符解码：检查当前字符 s[i-1] 是否为 '0'
            if (s.charAt(i - 1) != '0') {
                dp[i] += dp[i - 1];
            }

            // 双字符解码：检查 s[i-2] 和 s[i-1] 组成的两位数是否在 [10, 26] 范围内
            int twoDigit = Integer.parseInt(s.substring(i - 2, i));
            if (twoDigit >= 10 && twoDigit <= 26) {
                dp[i] += dp[i - 2];
            }
        }

        return dp[n];
    }
}
