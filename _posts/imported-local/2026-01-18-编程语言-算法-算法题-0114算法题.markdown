编写一个程序，通过填充空格来解决数独问题。
数独的解法需 遵循如下规则：
数字 1-9 在每一行只能出现一次。
数字 1-9 在每一列只能出现一次。
数字 1-9 在每一个以粗实线分隔的 3x3 宫内只能出现一次。（请参考示例图）
数独部分空格内已填入了数字，空白格用 '.' 表示。


输入：
board = [["5","3",".",".","7",".",".",".","."],["6",".",".","1","9","5",".",".","."],[".","9","8",".",".",".",".","6","."],["8",".",".",".","6",".",".",".","3"],["4",".",".","8",".","3",".",".","1"],["7",".",".",".","2",".",".",".","6"],[".","6",".",".",".",".","2","8","."],[".",".",".","4","1","9",".",".","5"],[".",".",".",".","8",".",".","7","9"]]
输出：
board = [["5","3","4","6","7","8","9","1","2"],["6","7","2","1","9","5","3","4","8"],["1","9","8","3","4","2","5","6","7"],["8","5","9","7","6","1","4","2","3"],["4","2","6","8","5","3","7","9","1"],["7","1","3","9","2","4","8","5","6"],["9","6","1","5","3","7","2","8","4"],["2","8","7","4","1","9","6","3","5"],["3","4","5","2","8","6","1","7","9"]]


我的思路： 我会使用回溯算法。主函数从 (0, 0) 开始调用 solve 递归函数。solve(board) 的含义是：尝试解决从当前 board 状态开始的数独。递归函数会找到第一个空格，然后依次尝试填入 '1' 到 '9'。对于每个尝试的数字，如果它符合数独规则（不与同行、同列、同宫的数字冲突），就将其填入，并递归调用 solve。如果递归调用返回 true，说明后续的格子都能被成功填充，那么整个数独解决了，返回 true。如果所有数字 1-9 都尝试失败，说明当前路径无解，需要撤销当前填入的数字，返回 false 进行回溯。



class Solution {
    public void solveSudoku(char[][] board) {
        if (board == null || board.length == 0 || board[0].length == 0) {
            return;
        }
        dfs(board, new char[]{'1', '2', '3', '4', '5', '6', '7', '8', '9'});
    }

    public boolean dfs (char[][] board,  char[] DIRECTIONS) {
        for(int i = 0; i < board.length; i++) {
            for (int j = 0; j < board[0].length; j++) {
                if (board[i][j] == '.') {
                    for (char ch: DIRECTIONS) {
                        if (solve(board, ch, i , j)) { // 1. 修正：添加了缺失的 )
                            board[i][j] = ch;
                            if(dfs(board, DIRECTIONS)) {
                                return true;
                            }
                            board[i][j] = '.';
                        }
                    }
                    // 2. 修正：将 return false 移动到内层 for 循环之后
                    return false;
                }
            }
        }
        // 3. 修正：添加 return true，表示数独已解决
        return true;
    }

    public boolean solve (char[][] board, char ch, int i, int j) {
        // 检查行
        for (int x = 0; x < board[0].length; x++) {
            if (board[i][x] == ch) {
                return false;
            }
        }

        // 检查列
        for (int x = 0; x < board.length; x++) {
            if (board[x][j] == ch) {
                return false;
            }
        }

        // 检查 3x3 宫格
        int startRow = (i / 3) * 3; // 修正：计算起始行
        int startCol = (j / 3) * 3; // 修正：计算起始列

        // 修正：使用新的循环变量 r, c，并使用正确的范围
        for(int r = startRow; r < startRow + 3; r++) { 
            for (int c = startCol; c < startCol + 3; c++){
                if (board[r][c] == ch){ // 修正：条件改为 &&
                    return false;
                }
            }
        }
        return true;
    }
}
