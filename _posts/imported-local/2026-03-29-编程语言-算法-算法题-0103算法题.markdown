# 2. 两数相加

> LeetCode 2 - Add Two Numbers

## 题目描述

给你两个 非空 的链表，表示两个非负整数。它们的数字逆序存储，并且每个节点包含一个数字。
请你将这两个数相加，并以相同形式返回一个表示和的链表。
你可以假设除了数字 0 之外，这两个数都不会以 0 开头。



**解题思路**

把两个链表的每一位（逆序存储）相加，维护一个进位 `carry`，生成新的链表即可。  
遍历时：

1. 取出当前结点的值（若链表已经遍历完则取 0）。  
2. 计算 `sum = val1 + val2 + carry`。  
3. `carry = sum // 10`，新结点的值为 `sum % 10`。  
4. 移动到下一个结点，继续处理，直到两条链表都走完并且 `carry == 0`。

时间复杂度：`O(max(m, n))`（`m、n` 为两条链表的长度），因为只需要遍历一次。  
空间复杂度：`O(max(m, n))`，用于存放结果链表。

## Python 实现（Python 3）

```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


def addTwoNumbers(l1: ListNode, l2: ListNode) -> ListNode:
    """返回两个链表相加得到的新链表（逆序）"""
    dummy = ListNode(0)          # 虚拟头结点，方便统一处理
    cur = dummy
    carry = 0

    while l1 or l2 or carry:     # 只要还有数字或进位就继续
        v1 = l1.val if l1 else 0
        v2 = l2.val if l2 else 0

        total = v1 + v2 + carry
        carry = total // 10
        cur.next = ListNode(total % 10)

        # 移动指针
        cur = cur.next
        l1 = l1.next if l1 else None
        l2 = l2.next if l2 else None

    return dummy.next


# ------------------- 测试 -------------------
def build_list(arr):
    """把列表 [2,4,3] 转成 2->4->3 的链表"""
    dummy = ListNode(0)
    cur = dummy
    for v in arr:
        cur.next = ListNode(v)
        cur = cur.next
    return dummy.next


def print_list(node):
    """把链表转成普通列表打印，方便查看结果"""
    res = []
    while node:
        res.append(node.val)
        node = node.next
    print(res)


# 示例：342 + 465 = 807 -> [7,0,8]
l1 = build_list([2, 4, 3])
l2 = build_list([5, 6, 4])
res = addTwoNumbers(l1, l2)
print_list(res)   # 输出: [7, 0, 8]
```

**说明**  
- `dummy` 用来简化链表头部的创建，最终返回 `dummy.next`。  
- `while l1 or l2 or carry` 确保即使最高位产生额外进位（如 `5 + 5 = 10`）也能正确生成结点。

## C++ 实现（C++17）

```cpp
struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(nullptr) {}
};

class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode dummy(0);
        ListNode* cur = &dummy;
        int carry = 0;

        while (l1 || l2 || carry) {
            int v1 = l1 ? l1->val : 0;
            int v2 = l2 ? l2->val : 0;

            int sum = v1 + v2 + carry;
            carry = sum / 10;
            cur->next = new ListNode(sum % 10);
            cur = cur->next;

            if (l1) l1 = l1->next;
            if (l2) l2 = l2->next;
        }
        return dummy.next;
    }
};
```

### 小结
- 关键点在于**逆序存储**，这正好可以直接从低位到高位逐位相加，无需翻转链表。  
- 进位 `carry` 需要在每一步都参与运算，并在最后检查是否仍有残余。  
- 代码实现非常简洁，时间和空间复杂度均线性，符合题目要求。祝你在面试或刷题中顺利通过！


/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        // 创建一个虚拟头节点，方便处理第一个节点的创建
        ListNode dummyHead = new ListNode(-1);
        // 当前操作的节点，从虚拟头节点开始
        ListNode current = dummyHead;
        // 进位值，初始为 0
        int carry = 0;

        // 循环继续的条件：l1 或 l2 还有节点，或者还有进位未处理
        while (l1 != null || l2 != null || carry != 0) {
            // 获取当前 l1 节点的值，如果 l1 已经遍历完，则值为 0
            int val1 = (l1 != null) ? l1.val : 0;
            // 获取当前 l2 节点的值，如果 l2 已经遍历完，则值为 0
            int val2 = (l2 != null) ? l2.val : 0;

            // 计算当前位的和（包括上一位的进位）
            int sum = val1 + val2 + carry;

            // 更新进位（整除 10）
            carry = sum / 10;

            // 创建新节点，存储当前位的值（对 10 取余）
            current.next = new ListNode(sum % 10);

            // 移动 current 指针到新创建的节点
            current = current.next;

            // 移动 l1 和 l2 指针到下一个节点（如果存在）
            if (l1 != null) l1 = l1.next;
            if (l2 != null) l2 = l2.next;
        }

        // 循环结束后，dummyHead.next 就是结果链表的第一个真实节点
        return dummyHead.next;
    }
}
