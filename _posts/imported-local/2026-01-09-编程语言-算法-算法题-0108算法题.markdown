1、给你一个整数数组 nums，判断是否存在三个元素 a、b、c，使得 a + b + c = 0？请你找出所有和为 0 且不重复的三元组。
示例 1：
输入: nums = [-1,0,1,2,-1,-4]
输出: [[-1,-1,2],[-1,0,1]]
解释: nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0。
nums[0] + nums[1] =  -nums[2] 

nums[0] + nums[2] + nums[3] = (-1) + 1 + 2 = 0。
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0。
nums[0] + nums[1] + nums[5] = (-1) + 0 + (-1) = -2。
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0。
nums[1] + nums[3] + nums[5] = 0 + 2 + (-1) = 1。
nums[2] + nums[3] + nums[4] = 1 + 2 + (-1) = 2。
nums[2] + nums[4] + nums[5] = 1 + (-1) + (-1) = -1。
nums[3] + nums[4] + nums[5] = 2 + (-1) + (-1) = 0。
