给你一个只包含 '(' 和 ')' 的字符串，找出最长有效（格式正确且连续）括号子串的长度。


💡 示例
示例 1：
输入： s = "(()"
输出： 2
解释： 最长有效括号子串是 "()"
示例 2：
输入： s = ")()())"
输出： 4
解释： 最长有效括号子串是 "()()"


栈（Stack）的应用
动态规划（Dynamic Programming）思想
字符串处理技巧
对括号匹配逻辑的深刻理解


import java.util.Stack;

class Solution {
    public int longestValidParentheses(String s) {
        if (s == null || s.length() <= 1) {
            return 0;
        }

        // 修正1: 使用 Stack<Integer> 来存储索引
        Stack<Integer> stack = new Stack<>();
        // 修正2: 预处理，压入哨兵 -1
        stack.push(-1);

        int maxLength = 0;

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);
            if (c == '(') {
                // 修正3: 压入的是索引 i
                stack.push(i);
            } else { // c == ')'
                // 弹出一个元素（可能是 '(' 的索引，也可能是哨兵或未匹配的 ')' 的索引）
                stack.pop();

                // 修正4: 计算有效长度
                if (stack.isEmpty()) {
                    // 如果栈为空，说明当前的 ')' 没有匹配项，它成为新的哨兵
                    stack.push(i);
                } else {
                    // 如果栈不为空，说明匹配成功，计算当前有效长度
                    // 长度 = 当前索引 i - 新的栈顶索引（即上一个未匹配位置的索引）
                    int currentLength = i - stack.peek();
                    maxLength = Math.max(maxLength, currentLength);
                }
            }
        }

        return maxLength;
    }
}
