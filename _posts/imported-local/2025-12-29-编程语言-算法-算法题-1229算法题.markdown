题目描述
给你一个整数数组 nums。如果数组中存在任意两个不同的索引 i 和 j，使得 nums[i] == nums[j]，则返回 true；否则返回 false。

class Solution {
    public boolean containsDuplicate(int[] nums) {
        Set<Integer> seen = new HashSet<>();
        for (int num : nums) {
            if (!seen.add(num)) {  // add() 返回 false 表示已存在
                return true;
            }
        }
        return false;
    }
}
