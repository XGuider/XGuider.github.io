题目编号： LeetCode 91
题目名称： 解码方法
难度： 🟠 中等
标签： 字符串、动态规划


一条包含字母 A-Z 的消息通过以下映射进行了 编码：
'A' -> "1"
'B' -> "2"
...
'Z' -> "26"
要 解码 已编码的消息，所有数字必须基于上述映射的方法，反向映射回字母（可能有多种方法）。例如，"11106" 可以映射为：
"AAJF"，将消息分组为 (1 1 10 6)
"KJF"，将消息分组为 (11 10 6)
注意，消息不能分组为 (1 11 06)，因为 "06" 不能映射为 "F"，这是由于 "6" 和 "06" 在映射中并不等价。
给你一个非空字符串 s，由数字组成，其中包含 '0'。请计算并返回 解码 方法的 总数。
题目数据保证答案肯定是一个 32 位的整数。


示例 1：
输入： s = "12"
输出： 2
解释： 它可以解码为 "AB"（1 2）或者 "L"（12）。

// s = "226"

class Solution {
    public int numDecodings(String s) {
        if (s == null || s.length() == 0 || s.charAt(0) == '0') {
            return 0; // 空字符串或以 0 开头都无法解码
        }

        int n = s.length();
        // dp[i] 表示 s 的前 i 个字符的解码方法数
        int[] dp = new int[n + 1];
        
        // 初始状态
        dp[0] = 1; // 空字符串有 1 种解码方式
        dp[1] = 1; // 第一个字符不为 '0' (已在 if 中检查)，所以有 1 种方式

        for (int i = 2; i <= n; i++) {
            // 检查当前字符 s[i-1] 是否可以单独解码
            if (s.charAt(i - 1) != '0') {
                dp[i] += dp[i - 1]; // 可以单独解码，方案数加上 dp[i-1]
            }

            // 检查前一个字符 s[i-2] 和当前字符 s[i-1] 是否可以联合解码
            int twoDigit = Integer.parseInt(s.substring(i - 2, i));
            if (twoDigit >= 10 && twoDigit <= 26) {
                dp[i] += dp[i - 2]; // 可以联合解码，方案数加上 dp[i-2]
            }
            // 如果以上两种情况都不满足，dp[i] 保持为 0，这符合预期。
        }

        return dp[n];
    }
}
