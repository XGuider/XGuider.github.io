标签： 数组、动态规划

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。
给定一个代表每个房屋存放金额的非负整数数组 nums，计算你不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。



输入： nums = [1,2,3,1]
输出： 4
解释： 偷窃 1 号房屋 (金额 = 1) ，然后偷窃 3 号房屋 (金额 = 3)。
偷窃到的最高金额 = 1 + 3 = 4 。


class Solution {
	public int maxProfit(int[] nums){
		int[] dp = new int();
		if (nums.length = 1) {dp[0] = nums[0]; return dp[0];}
		if (nums.length = 2) {dp[0] = Math.max(nums[0], nums[2]);return dp[1];}

		for(int i = 2; i < nums.length; i++) {
			dp[i] = Math.max(dp[i-2] + nums[i], dp[i-1]);
		}
		return dp[nums.length-1];
	}
}


======标准答案======
class Solution {
    public int rob(int[] nums) { // 1. 修正：方法名改为 rob
        if (nums == null || nums.length == 0) {
            return 0; // 处理空数组的情况
        }
        
        int n = nums.length;
        if (n == 1) {
            return nums[0]; // 2. 修正：直接返回 nums[0]
        }
        
        // 3. 修正：正确初始化 dp 数组
        int[] dp = new int[n]; 

        dp[0] = nums[0];
        // 4. 修正：正确计算 dp[1]
        dp[1] = Math.max(nums[0], nums[1]); 

        // 5. 修正：从 i=2 开始循环
        for(int i = 2; i < n; i++) { 
            dp[i] = Math.max(dp[i-2] + nums[i], dp[i-1]);
        }
        
        return dp[n-1];
    }
}
