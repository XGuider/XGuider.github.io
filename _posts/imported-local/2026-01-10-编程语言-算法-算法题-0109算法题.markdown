题目编号： LeetCode 5
题目名称： 最长回文子串
给你一个字符串 s，找到 s 中最长的回文子串。
回文（Palindrome）是指正着读和反着读都一样的字符串。例如："aba", "abccba"。
子串（Substring）是连续的字符序列。


例如：
我的思路： 我会用中心扩散法。回文串有两种类型：奇数长度（如 "aba"）和偶数长度（如 "abba"）。我将遍历字符串，以每个字符为中心（奇数）和以每两个相邻字符之间为中心（偶数）进行扩散，记录最长的回文串。


class Solution {
    public String longestPalindrome(String s) {
        if (s == null || s.length() < 1) {
            return "";
        }

        int start = 0;
        int maxLen = 0;

        for (int i = 0; i < s.length(); i++) {
            // 以 i 为中心的奇数长度回文
            int len1 = expandAroundCenter(s, i, i);
            // 以 i 和 i+1 之间的间隙为中心的偶数长度回文
            int len2 = expandAroundCenter(s, i, i + 1);

            // 取两者中较长的
            int currentMaxLen = Math.max(len1, len2);

            // 如果当前找到的回文更长，则更新 start 和 maxLen
            if (currentMaxLen > maxLen) {
                maxLen = currentMaxLen;
                // 计算回文的起始索引
                // 当前中心是 i 和 i 或 i 和 i+1
                // 回文长度为 maxLen，所以起始索引 = i - (maxLen - 1) / 2
                start = i - (currentMaxLen - 1) / 2;
            }
        }

        return s.substring(start, start + maxLen);
    }

    /**
     * 以 left 和 right 为中心向两边扩散，返回回文串的长度
     */
    private int expandAroundCenter(String s, int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }
        // 循环结束时，left 和 right 分别指向不匹配的字符
        // 所以实际回文长度是 right - left - 1
        return right - left - 1;
    }
}
