题目：寻找数组中三个数的最大乘积
题目描述：
给定一个整数数组 nums，请你在数组中找出三个数，使得它们的乘积最大，并返回这个最大乘积。

示例：
输入：nums = [5, -2, 3, 1, -4]
输出：60
解释：最大的乘积是 5 * (-4) * (-2) = 60。
输入：nums = [-1, -2, -3]
输出：-6
解释：在这种情况下，三个负数相乘结果为负，(-1) * (-2) * (-3) = -6 是最大的乘积。


class Solution {
	public int maximumProduct(int[] nums) {
		if (nums == null || nums.length < 3) {return 0;}
		Arrays.sort(nums);
		int n = nums.length;
		int max01=nums[n-1]*nums[n-2]*nums[n-3];
		int max02=nums[n-1]*nums[n-2]*nums[0];

		return Math.max(max01, max02);
	}
}



import java.util.Arrays;

class Solution {
    public int maximumProduct(int[] nums) {
        if (nums == null || nums.length < 3) {
            return 0; // 或者根据题目要求抛出异常
        }
        
        Arrays.sort(nums); // 修正1: 导入Arrays并正确调用
        int n = nums.length;
        
        // 修正2: 正确定义变量
        int max01 = nums[n-1] * nums[n-2] * nums[n-3]; // 三个最大数的乘积
        int max02 = nums[n-1] * nums[0] * nums[1];     // 一个最大数和两个最小数的乘积
        
        // 修正3: 使用Math.max并添加分号
        return Math.max(max01, max02);
    }
}
