如果用户第一天登录算新用户，求每一天登录的新老用户数
![[file-20260331203359657.png]]

``
`WITH new_user AS ( -- 1. 找出每个用户的首次登录日期 SELECT user_id, MIN(login_date) AS first_login_date FROM A GROUP BY user_id ) 

`-- 2. 主查询：统计每日的新老用户 SELECT A.login_date, COUNT(B.user_id) AS new_users, -- 匹配成功的即为新用户 COUNT(A.user_id) - COUNT(B.user_id) AS old_users -- 总数减去新用户即为老用户 FROM A LEFT JOIN new_user B ON A.user_id = B.user_id AND A.login_date = B.first_login_date -- 只有当登录日期等于首次登录日期时才匹配 GROUP BY A.login_date ORDER BY A.login_date;
`
