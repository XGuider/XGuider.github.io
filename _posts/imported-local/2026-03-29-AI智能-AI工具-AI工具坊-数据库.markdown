# 数据库设计文档

> 用户表与访问统计表的 SQL 设计

## 概述

本文档记录用户表和访问统计表的数据库设计，包含建表语句、常用操作和维护命令。

---

## 一、用户表（user）

### 表结构

```sql
CREATE TABLE `user` (
  `id` bigint NOT NULL AUTO_INCREMENT,
  `open_id` varchar(100) DEFAULT NULL COMMENT '微信OpenID',
  `union_id` varchar(100) DEFAULT NULL COMMENT '微信UnionID',
  `nickname` varchar(100) DEFAULT NULL COMMENT '微信昵称',
  `avatar` varchar(500) DEFAULT NULL COMMENT '微信头像',
  `level` enum('GUEST','MEMBER','VIP','ADMIN') DEFAULT 'GUEST' COMMENT '用户等级',
  `vip_expire_time` datetime DEFAULT NULL COMMENT 'VIP到期时间',
  `today_visit_count` int DEFAULT '0' COMMENT '今日访问次数',
  `last_visit_time` datetime DEFAULT NULL COMMENT '最后访问时间',
  `status` enum('NORMAL','BANNED') DEFAULT 'NORMAL' COMMENT '用户状态',
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `update_time` datetime DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  `password` varchar(255) DEFAULT NULL,
  PRIMARY KEY (`id`),
  UNIQUE KEY `nickname` (`nickname`),
  KEY `idx_open_id` (`open_id`),
  KEY `idx_level` (`level`),
  KEY `idx_status` (`status`)
) ENGINE=InnoDB AUTO_INCREMENT=10 DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci COMMENT='用户表';
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键，自增 |
| open_id | varchar(100) | 微信 OpenID |
| union_id | varchar(100) | 微信 UnionID |
| nickname | varchar(100) | 微信昵称，唯一 |
| avatar | varchar(500) | 头像 URL |
| level | enum | 用户等级：GUEST/MEMBER/VIP/ADMIN |
| vip_expire_time | datetime | VIP 到期时间 |
| today_visit_count | int | 今日访问次数 |
| last_visit_time | datetime | 最后访问时间 |
| status | enum | 用户状态：NORMAL/BANNED |
| password | varchar(255) | 密码（管理员用） |

## 二、访问统计表（visit_stats）

### 表结构

```sql
CREATE TABLE `visit_stats` (
  `id` bigint NOT NULL AUTO_INCREMENT,
  `user_id` bigint NOT NULL,
  `ad_clicks` int DEFAULT NULL,
  `comment_count` int DEFAULT NULL,
  `create_time` datetime(6) DEFAULT NULL,
  `resource_visits` int DEFAULT NULL,
  `stat_date` date DEFAULT NULL,
  `total_visits` int DEFAULT NULL,
  `unique_visitors` int DEFAULT NULL,
  `update_time` datetime(6) DEFAULT NULL,
  `vip_conversions` int DEFAULT NULL,
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| user_id | bigint | 用户 ID |
| ad_clicks | int | 广告点击数 |
| comment_count | int | 评论数 |
| stat_date | date | 统计日期 |
| total_visits | int | 总访问量 |
| unique_visitors | int | 独立访客数 |
| resource_visits | int | 资源访问数 |
| vip_conversions | int | VIP 转化数 |

## 三、常用操作

### 修改字段

```sql
-- 修改 level 字段
ALTER TABLE user MODIFY COLUMN level enum('GUEST','MEMBER','VIP', 'ADMIN') DEFAULT 'GUEST' COMMENT '用户等级';

-- 修改 open_id 字段长度
ALTER TABLE user MODIFY COLUMN `open_id` varchar(100) DEFAULT NULL COMMENT '微信OpenID';
```

### 更新数据

```sql
-- 将 nickname 为 'root' 的用户设为管理员
UPDATE user SET level = 'ADMIN' WHERE nickname = 'root';

-- 将 id=2 的资源设为 VIP 区
UPDATE resource SET section = 'VIP' WHERE id = 2;
```

### 删除数据

```sql
-- 删除指定用户
DELETE FROM user WHERE id = 10;
```

### 索引操作

```sql
-- 删除旧索引
ALTER TABLE user DROP INDEX open_id;

-- 添加新唯一键
ALTER TABLE user ADD UNIQUE KEY nickname;
```

## 四、用户等级说明

| 等级 | 说明 | 权限 |
|------|------|------|
| GUEST | 游客 | 每日 3 次下载 |
| MEMBER | 普通会员 | 无限普通资源 |
| VIP | 超级会员 | 全部资源 + VIP 资源 |
| ADMIN | 管理员 | 后台管理权限 |
