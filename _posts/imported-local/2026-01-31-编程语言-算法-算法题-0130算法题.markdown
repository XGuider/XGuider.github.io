题目编号： LeetCode 122
题目名称： 买卖股票的最佳时机 II
难度： 🟢 简单
标签： 数组、动态规划、贪心


给你一个整数数组 prices，其中 prices[i] 表示某支股票第 i 天的价格。
在每一天，你可以决定是否购买和/或出售股票。你在任何时候 最多 只能持有 一股 股票。你也可以先购买，然后在 同一天 出售。
返回你能获得的最大利润。
💡 示例
示例 1：
输入： prices = [7,1,5,3,6,4]// [7,1,2,3,8,4]
输出： 7
解释： 在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 获得利润 4。随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 获得利润 3。总利润为 4 + 3 = 7。

dp[n-1] = dp[]


class Solution {
    public int maxProfit(int[] prices) {
        if (prices == null || prices.length <= 1) {
            return 0;
        }

        int totalProfit = 0;

        // 从第二天开始，计算每天相对于前一天的利润
        for (int i = 1; i < prices.length; i++) {
            int dailyProfit = prices[i] - prices[i - 1];
            // 如果今天比昨天涨了，就把这个利润加上
            if (dailyProfit > 0) {
                totalProfit += dailyProfit;
            }
        }

        return totalProfit;
    }
}

class Solution {
    public int maxProfit(int[] prices) {
        if (prices == null || prices.length <= 1) {
            return 0;
        }

        int n = prices.length;
        // dp[i][0] 表示第 i 天不持股的最大利润
        // dp[i][1] 表示第 i 天持股的最大利润
        int[][] dp = new int[n][2];

        // 初始状态
        dp[0][0] = 0;                 // 第一天不持股，利润为 0
        dp[0][1] = -prices[0];        // 第一天持股，利润为 -prices[0]

        // 从第二天开始遍历
        for (int i = 1; i < n; i++) {
            // 今天不持股 = max(昨天就不持股, 昨天持股今天卖掉)
            dp[i][0] = Math.max(dp[i-1][0], dp[i-1][1] + prices[i]);
            // 今天持股 = max(昨天就持股, 昨天不持股今天买入)
            dp[i][1] = Math.max(dp[i-1][1], dp[i-1][0] - prices[i]);
        }

        // 最后一天不持股的利润是最大的
        return dp[n-1][0];
    }
}


class Solution {
    public int maxProfit(int[] prices) {
        if (prices == null || prices.length <= 1) {
            return 0;
        }

        int n = prices.length;
        // sold 代表当前天不持股的最大利润
        int sold = 0;
        // hold 代表当前天持股的最大利润
        int hold = -prices[0];

        for (int i = 1; i < n; i++) {
            // 临时变量保存旧的 sold 值，因为计算新的 hold 会用到旧的 sold
            int prevSold = sold;
            
            // 更新 sold: sold = max(昨天就不持股, 昨天持股今天卖掉)
            sold = Math.max(sold, hold + prices[i]);
            // 更新 hold: hold = max(昨天就持股, 昨天不持股今天买入)
            hold = Math.max(hold, prevSold - prices[i]);
        }

        return sold;
    }
}
