题目编号： LeetCode 139
题目名称： 单词拆分
难度： 🟠 中等
标签： 字符串、动态规划、哈希表


给你一个字符串 s 和一个字符串列表 wordDict 作为字典。如果可以利用字典中出现的一个或多个单词拼接出 s，就返回 true。
注意：不要求字典中出现的单词全部都使用，并且字典中的单词可以重复使用。


示例 1：
输入: s = "leetcode", wordDict = ["leet", "code"]
输出: true
解释: 返回 true 因为 "leetcode" 可以由 "leet" 和 "code" 拼接而成。

示例 2：
输入: s = "applepenapple", wordDict = ["apple", "pen"]
输出: true
解释: 返回 true 因为 "applepenapple" 可以由 "apple", "pen", "apple" 拼接而成。注意，你可以重复使用字典中的单词。

示例 3：
输入: s = "catsandog", wordDict = ["cats", "dog", "sand", "and", "cat"]
输出: false

import java.util.*;

class Solution {
    public boolean wordBreak(String s, String[] wordDict) {
        // 1. 预处理：将 wordDict 存入 HashSet 以提高查找速度
        Set<String> wordSet = new HashSet<>(); // 修正1: 使用 Set<String> 而非 HashMap
        for (String word : wordDict) { // 使用增强for循环更简洁
            wordSet.add(word);
        }

        int n = s.length();
        // 2. 状态定义：dp[i] 表示 s 的前 i 个字符能否被拆分
        boolean[] dp = new boolean[n + 1];

        // 3. 初始状态：空字符串可以被拆分
        dp[0] = true;

        // 4. 状态转移：外层循环遍历字符串的每个位置
        for (int i = 1; i <= n; i++) {
            // 内层循环寻找分割点 j
            for (int j = 0; j < i; j++) {
                // 如果 s[0:j] 可以拆分，且 s[j:i] 是一个字典中的单词
                if (dp[j] && wordSet.contains(s.substring(j, i))) {
                    dp[i] = true; // 那么 s[0:i] 也可以拆分
                    break; // 找到一个可行解即可跳出内层循环
                }
            }
        }

        // 5. 最终结果：dp[n] 表示整个字符串 s 是否可以被拆分
        return dp[n];
    }
}
