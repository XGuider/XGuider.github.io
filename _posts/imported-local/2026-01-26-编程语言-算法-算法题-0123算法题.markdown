一个机器人位于一个 m x n 网格的左上角（起始点在下图中标记为 “Start” ）。
机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。
问总共有多少条不同的路径？


示例 1：
输入： m = 3, n = 7
输出： 28
示例 2：
输入： m = 3, n = 2
输出： 3
解释：
从左上角开始，总共有 3 条路径可以到达右下角。
向右 -> 向下 -> 向下
向下 -> 向右 -> 向下
向下 -> 向下 -> 向右



class Solution {
	public int uniquePaths(int m, int n) {

		if (m == 0 || n == 0) {return 0;}
		int[][] dp = new int[m][n];
		for (int i = 0; i < m; i++) {
			for (int j = 0; j < n; j++) {
				if (i == 0) {
					dp[0][j] = 1;
				} 
				else if (j == 0) {
					dp[i][0] = 1;

				} else {
					dp[i][j] = dp[i][j-1] + dp[i-1][j];
				}
			}
		}
		return dp[m-1][n-1];	
	}
}



class Solution {
    public int uniquePaths(int m, int n) {
        // 创建一维 DP 数组，并初始化为 1
        // 这相当于初始化了二维 DP 的第一行
        int[] dp = new int[n];
        for (int j = 0; j < n; j++) {
            dp[j] = 1;
        }

        // 从第二行开始迭代 (i=1 对应原来的 i=1 行)
        for (int i = 1; i < m; i++) {
            // 从第二列开始迭代 (j=1 对应原来的 j=1 列)
            // j=0 的列始终是 1 (对应二维 DP 的第一列)，无需更新
            for (int j = 1; j < n; j++) {
                // dp[j] (新) = dp[j] (旧，代表上一行的 dp[i-1][j]) + dp[j-1] (代表当前行左边的 dp[i][j-1])
                dp[j] = dp[j] + dp[j - 1];
            }
        }
        
        // dp[n-1] 现在存储的就是到达右下角的路径总数
        return dp[n - 1];
    }
}
